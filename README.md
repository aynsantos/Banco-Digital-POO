## Banco Digital POO

Abstraindo o conhecimento bancário, resumi uma solução orientada a objetos em Java. 

## Features:

O banco oferece `ContaCorrente` `ContaPoupança`.<br>
Funcionalidades `Sacar`, `Depositar` e `Transferir`.<br>
